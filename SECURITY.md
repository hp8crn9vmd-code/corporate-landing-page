# Security Policy

## Supported Versions
This project is maintained as a static site. Security updates are applied to
the latest version of the repository only.

## Reporting a Vulnerability
If you discover a security vulnerability, please report it responsibly.

- Do NOT open a public issue describing the vulnerability.
- Contact the project maintainer privately with detailed information.
- Include steps to reproduce the issue if possible.

## Scope
This project:
- Does not include a backend or server-side APIs
- Does not store user data
- Uses client-side forms via Web3Forms

## Security Measures
- Static export (no runtime server)
- Content Security Policy (CSP) applied
- No API Routes or server secrets
- Regular dependency updates via CI

## Disclosure Policy
Reported vulnerabilities will be reviewed and addressed as quickly as possible.
Responsible disclosure is appreciated.
